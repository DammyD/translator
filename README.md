# Translation App
The Translation App is a web application designed to assist users in translating text into various languages using OpenAI's language processing capabilities. Inspired by popular language learning platform **Duolingo**, this app aims to provide users with quick and accurate translations to aid their language learning journey.

## Features
- Translate text into multiple languages.

- Utilizes OpenAI's powerful language processing models.

- User-friendly interface inspired by popular language learning apps.

- Built with modern web technologies: React, Vite, and OpenAI.

## Installation
1. Clone the repository:

`git clone https://github.com/DammyD/translator.git`

2. Navigate to the project directory:

`cd translator-app`

3. Install dependencies:

`npm install`

## Usage
1. Start the development server:

`npm run dev`

2. Open your web browser and navigate to `https://localhost:5173` to access the Translator.

Enter the text you want to translate and select the desired target language from the list of available languages.

Click the *Translate* button to view the translated text.

## Technologies Used
**React:** A JavaScript library for building user interfaces.

**Vite:** A fast, modern build tool for web development.

**OpenAI:** Provides powerful language processing models for natural language understanding and generation.
